# Sistema de Gestión de Garaje 

Aplicación de escritorio desarrollada en Python utilizando Tkinter para registrar vehículos que ingresan a un garaje.

## Funcionalidades

- Registrar vehículos
- Visualizar vehículos registrados
- Limpiar los campos del formulario
